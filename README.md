# docker-fuck

# How To
## Build docker container
```
docker build -t kaliprog .
```
or
```
docker build -t kaliwork .
```

## Stardup container
```
docker run -d -it kaliprog 
```
or
```
docker run -d -it kaliwork
```

## Attach to container
```
docker attach kaliprog
```
or
```
docker attach kaliwork
```
